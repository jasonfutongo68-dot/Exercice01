# Exercice 01 - Trouver le maximum de trois nombres

## Description
Ce programme Python permet à l'utilisateur de saisir trois nombres et affiche le plus grand (le maximum) parmi eux.

## Objectif de l'exercice
- Pratiquer la saisie utilisateur avec `input()`
- Convertir des chaînes de caractères en nombres flottants avec `float()`
- Utiliser la fonction intégrée `max()` pour comparer des valeurs
- Appliquer les bases de l'affichage formaté avec les f-strings
